Key Design Decisions

PyodideProvider (React Context)

Loads Pyodide once via a web worker (keeps UI responsive during 3-8s cold load). Exposes a useCalculator() hook that provides evaluate(expr) and convert(qty, unit). All components access the engine through this hook — no prop drilling.

Constants/Units as static JSON + Pyodide

The constants and units tables are populated from a bundled JSON file (fast, no WASM needed for browsing). But numeric values are computed live by Pyodide when a constant is used in an expression — no drift between displayed data and calculation results.

Implicit multiplication preserved

The existing m_e c^2 syntax works unchanged. The expression is sent to Pyodide which runs the existing simple_eval / sympy parser. No new parsing code to write.

SearchBar

Global search across constants, units, and equation templates. Matches against both symbol ("G") and physical name ("gravitational constant", "Planck", "proton mass"). Results update on each keystroke (filters a preloaded JSON dataset, no Pyodide needed — instant).

Example: typing "energy" would match: Boltzmann constant (k_B, energy/temperature), Rydberg constant (Ryd, energy), eV (electronvolt), MeV, GeV, Stefan-Boltzmann law (template).

ConstantsTable

Scrollable table with columns: Symbol | Name | Value | Unit | Uncertainty. Clicking a row inserts the symbol into the expression editor at cursor position. Filtered by search. Data comes from a bundled JSON snapshot of all astropy constants (generated at build time).

UnitsTable

Grouped by category (Length, Mass, Time, Energy, etc.). Each row shows the unit name. Clicking inserts the unit into the editor. Simple reference — no value computation needed until the unit is used in an expression.

EquationTemplates

A preset list of common astrophysics equations (escape velocity, Schwarzschild radius, Kepler's 3rd law, Eddington luminosity, etc.). Each template shows:

  • The equation in readable form (rendered with KaTeX)
  • Named parameters with placeholder values
  • An "Add" button that inserts the full expression as a multi-line input: variable assignments for each parameter prefilled with default values, followed by the equation itself

Example — clicking "Add" on Escape Velocity inserts:

M = 1.4 M_sun
R = 10 km
sqrt(2 G M / R) in km/s

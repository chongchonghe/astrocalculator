Design — Section 4 (Revised): Equations Format v3

Each expression optionally carries its own latex string. Sidebar shows the LaTeX-rendered equation below the name, or falls back to a plain code string.

Updated Markdown File Format

--- title: Escape Velocity category: Mechanics tags: [velocity, gravity, compact objects] params: - symbol: M default: "1.4 M_sun" description: Mass of the object - symbol: R default: "10 km" description: Radius expressions: - name: "Escape velocity" expression: "sqrt(2 G M / R) in km/s" latex: "v_{\\rm esc} = \\sqrt{\\frac{2GM}{R}}" - name: "Circular velocity" expression: "sqrt(G M / R) in km/s" latex: "v_{\\rm circ} = \\sqrt{\\frac{GM}{R}}" description: "Circular orbital velocity, 1/√2 times escape velocity" - name: "Free-fall timescale" expression: "1 / sqrt(G rho)" description: "No LaTeX provided — renders as code" --- (optional markdown body — not parsed for sidebar display)

Which platform should the new AstroCalculator target?

The current version is a CLI tool. Each option below trades off different aspects of UX, portability, and development effort.

Browser-based calculator with a rich GUI, searchable constants browser, unit picker, and history sidebar. Accessible anywhere without install.

Key trade-offs

Web App

  • Zero install for users
  • Rich visual design possible
  • Easy to share/bookmark calculations
  • Works on any device

Web App

  • Requires hosting/deployment
  • Needs frontend + backend skills
  • Internet dependency

Desktop App

  • Full OS integration
  • Offline by default
  • brew install possible

Desktop App

  • Larger binary/install
  • Cross-platform testing burden
  • Electron is heavy

Enhanced TUI

  • Stays in terminal (astronomers' home)
  • Fastest to develop
  • Keep existing Python code
  • SSH-friendly

Enhanced TUI

  • Limited visual polish
  • Harder to browse constants
  • Mouse support limited
